## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Laksh25-code/laksh-portfolio.git
```

Go to project folder:

```bash
cd laksh-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
